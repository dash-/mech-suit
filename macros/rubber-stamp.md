# The Rubber Stamp (`rubber-stamp`)

An ethics gate or verification loop that approves everything. If it never blocks, it's not providing value — audit its criteria. **Corrective:** **Warm-up & Calibration** (backtest the gate against known cases) and **Decision Calibration** (review whether the gate adds value over time).
