# The Expensive Default (`expensive-default`)

Using Redundant Generation (5x cost) or Adversarial Review (5-15x cost) for low-stakes tasks that could be handled by Reflection (1.5x cost). **Corrective:** **Budget-Aware Execution** and **Confidence Signaling** (let confidence drive which quality patterns are applied).
