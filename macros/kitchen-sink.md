# The Kitchen Sink (`kitchen-sink`)

Applying every pattern to every task. Each pattern adds cost and latency. Use the minimum set needed for the quality and reliability requirements. **Corrective:** **Budget-Aware Execution** (force cost/importance tradeoffs) and **Escalation Ladder** (start cheap, escalate only when needed).
