# The Over-Decomposer (`over-decomposer`)

Breaking a task into so many subtasks that reassembly cost exceeds the benefit of decomposition. Some tasks are better handled holistically. **Corrective:** **Reflection** (try the task holistically with self-critique first, decompose only if that fails).
